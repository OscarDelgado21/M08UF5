Párrafo sobre los enlaces relativos: Las URL relativas prescinden de algunas partes
de las URL para hacerlas más breves. Como se trata de URL incompletas, es necesario
disponer de información adicional para obtener el recurso enlazado. En concreto,
para que una URL relativa sea útil es imprescindible conocer la URL del origen del enlace.
